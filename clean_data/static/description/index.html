Mass Clear Data(Clean Data)

This module helps user to clear multiple records of different objects from the wizard.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No configurations required.

Mass Clear Data(Clean Data)

To clean data records go to Settings > Clean Data > Clean Data

From this wizard,user can select one or more objects to delete records from.

Notes:
Do not use this module in production environment.
because using this module data clean/clear easily but restore is very difficult,
So do not use this module in production environment.
This module support only core object of the odoo not for custom objects.
